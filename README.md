# 筋トレLAB 🧪

1日◯分×週◯日で「どれだけ変わるか」を科学で予測する、登録不要の無料筋トレ設計アプリ。

**▶ https://eo8883494-png.github.io/kintore-lab/**

## 機能

- 🎯 **メニュー自動生成** — 環境(自宅/ダンベル/ジム)・経験・目標・狙う部位(ボディマップでタップ)から週間メニューを自動設計。サボった日の振替、やり残しの自動繰り越し付き
- ⚗️ **効率シミュレーター** — 週あたりセット数と筋肥大の用量反応関係に基づき、総合効率・月あたり筋肉増加・3ヶ月/半年/1年後を予測
- 🍚 **食事プラン** — 目標からPFCを計算し、実際の食品と量(ご飯150g、卵2個…)で献立を自動生成。体重ナビが「あと◯kg・週◯kgペース」まで提示
- ✍️ **記録** — チェックするだけの自動記録、推定1RM成長グラフ、週間ボリューム、ストリークカレンダー、体型フォト(端末内保存)
- 📸 **シェア** — トレ記録のシェア画像を自動生成。#筋トレLAB でみんなの記録とつながる
- 🧰 **ツール** — 休憩タイマー、1RM計算、タンパク質/カロリー計算、FFMI、超回復ガイド

## 技術

素のHTML/CSS/JS(ビルド無し・依存ゼロ・約200KB)。データは全てlocalStorage/IndexedDBで端末内保存、サーバー送信なし。種目DB(78種目)と食品DB(51品目)はマルチエージェントワークフローで生成・検証。

## 開発

```
python -m http.server 8137 -d .
```

JS/CSSを変更したら index.html のクエリ `?v=N` を必ず上げること(キャッシュバスト)。
`seed-demo.html` / `share-preview.html` は開発用(.gitignore済み・デプロイ禁止)。
